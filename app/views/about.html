About

DoggerJS is a frontend solution to create easyly a project using AngularJS, GulpJS and Bower.

DoggerJS already integrate most commons libraries used and needed for client side develpment.

Modern

DoggerJS is a solution for creating applications using modern web standards.

Powered by the Open Source Community

DoggerJS is built by you, for you. We do our work in the open so that you can either help DoggerJS improve or extend and modify what we've built.
